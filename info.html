Context

  • Physical exercise is an important part of people's lives, specially to those who are Personal Trainers;

  • With the COVID-19 pandemic, it became harder for people to go to the gym, and for PT's to do their job;

  • Given the technology available, it is possible for an application to be developed that provides guidance towards physical exercise, supported by remote communication.

Problem

  • Usually, applications that help with practising physical exercise at home do not create a line of communication between the users and one or several instructors;

  • If an app was created from the Personal Trainer's point of view, could the interaction between them and their clients be facilitated?

Objectives

  • Assist people practising physical exercise outside of a gym, with an instructor's insight in mind:
    • Create a database with multiple exercises, and their details;
    • Develop an API for the applications;
    • Develop mobile and desktop applications, with the possibility of remote communication between users and trainers.

Expected results

  • Mobile and desktop applications that allow:
    • The instructor to provide a service to their client;
    • Instructor-Client interaction;
    • Friendly competition between clients;
  • Integration of mobile devices (smartwatch, fitbit, etc.) with the applications.
